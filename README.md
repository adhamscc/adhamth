# موقع دعاء ذيابات الشخصي (Portfolio)

موقع شخصي ثنائي اللغة (عربي/إنجليزي) جاهز للرفع على GitHub Pages، يحتوي على 4 أقسام: السيرة الذاتية، المشاريع، طلب خدمة، والدورات التدريبية.

## 📁 هيكل الملفات

```
├── index.html          # الصفحة الرئيسية (كل الأقسام فيها)
├── css/style.css        # التصميم
├── js/main.js            # تبديل اللغة، القائمة، نموذج طلب الخدمة
└── assets/                # الصور والفيديوهات والـ CV
    ├── profile-placeholder.svg
    ├── project-placeholder-1/2/3.svg
    └── cv-doaa-thiabat.pdf   (يجب إضافته)
```

## 🚀 كيفية الرفع على GitHub Pages

1. أنشئي مستودع (Repository) جديد على GitHub، مثلاً باسم `portfolio` أو `USERNAME.github.io`.
2. ارفعي كل ملفات هذا المجلد كما هي (بنفس الهيكل).
3. من إعدادات المستودع Settings → Pages → اختاري Branch: `main` والمجلد `/root`، ثم Save.
4. بعد دقيقة أو دقيقتين، الموقع بيصير شغال على رابط مثل:
   `https://USERNAME.github.io/portfolio/`

## ✏️ التعديلات المطلوبة قبل النشر (المحتوى تجريبي حالياً)

كل نص بين قوسين `[ ]` هو محتوى تجريبي (Placeholder) لازم تستبدليه بمحتواك الحقيقي. ابحثي في `index.html` عن الأقواس المربعة وعدّليها.

### 1. الروابط الشخصية
في `index.html`، استبدلي:
- `USERNAME` في روابط GitHub وLinkedIn (يوجد في مكانين: Hero والـ Footer)
- رقم الواتساب في `js/main.js` (المتغير `whatsappNumber` في أعلى الملف) — اكتبيه بصيغة دولية بدون + أو 00، مثال: `962791234567`

### 2. السيرة الذاتية
- استبدلي النبذة، المهارات، الخبرات، والتعليم في قسم `#cv`
- ضيفي ملف PDF فعلي باسم `cv-doaa-thiabat.pdf` داخل مجلد `assets/` (زر "تحميل السيرة الذاتية" بيسحبه تلقائياً)

### 3. المشاريع
- لكل مشروع: استبدلي الاسم، الوصف، التقنيات المستخدمة
- استبدلي `project-placeholder-1/2/3.svg` بصور حقيقية لمشاريعك (نفس الأسماء أو غيّري المسار في `index.html`)
- لروابط الفيديو: أسهل طريقة هي رفع الفيديو على يوتيوب (Unlisted لو ما بدك يكون عام) وربط الزر "فيديو الشرح" برابط اليوتيوب مباشرة

### 4. طلب خدمة
النموذج جاهز ويشتغل بدون سيرفر: عند الضغط "إرسال عبر واتساب" بيفتح واتساب مع رسالة جاهزة، و"إرسال عبر الإيميل" بيفتح تطبيق الإيميل. فقط تأكدي من رقم الواتساب المذكور أعلاه.

### 5. الدورات التدريبية
عدّلي عناوين ووصف الدورات الثلاث في قسم `#courses` حسب الدورات الفعلية اللي بدك تعرضيها.

## 🌐 تبديل اللغة
الموقع يدعم عربي/إنجليزي بزر EN/AR في أعلى الصفحة. كل عنصر نص فيه `data-ar` و`data-en` — لو ضفتي محتوى جديد، تأكدي تضيفي له نفس الخاصيتين حتى يترجم صح.

## 🎨 تغيير الألوان
كل الألوان معرّفة في أول ملف `css/style.css` تحت `:root`. غيّري قيم `--primary` و`--accent` لتغيير الألوان الأساسية بسهولة بمكان واحد.
